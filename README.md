# COFFEE SALES DASHBOARD (Interactive Dashboard created using MS Excel)

## Project Objective:

Built a Coffee Sales Analytics Dashboard in Excel to track sales KPIs, identify customer trends, and support business decision-making using data visualization.

## DataSet Used:
- <a href="https://github.com/ankittanalytics1/COFFEE-SALES-DASHBOARD/blob/main/COFFEE%20SALES%20DASHBOARD.xlsx"> RAW DATASET </a>

Business Questions
- What is the total sales revenue generated? 
- How many customers visited the coffee shop (footfall)? 
- What is the average bill per customer? 
- What is the average number of items ordered per customer? 
- During which hours are the most orders placed? 
- Which product categories contribute the most to sales? 
- What are the top 5 best-selling products? 
- Which store location generates the highest sales and footfall? 
- How do sales and orders vary across weekdays? 
- Which cup size is ordered most frequently? 

- <a href="https://github.com/ankittanalytics1/COFFEE-SALES-DASHBOARD/blob/main/COFFEE%20SALES%20DASHBOARD.xlsx(https://github.com/ankittanalytics1/COFFEE-SALES-DASHBOARD/blob/main/SCREENSHOT.png)"> COFFEE SALES DASHBOARD </a>

## KPIs Tracked
•	Total Sales 
•	Total Footfall 
•	Average Bill per Person 
•	Average Order per Person 
•	Sales by Product Category 
•	Top 5 Products by Sales 
•	Sales by Store Location 
•	Orders by Hour 
•	Orders by Weekday 
•	Order Distribution by Size

## Process
•	Data Cleaning & Validation 
•	Pivot Table Creation

•	Dashboard Design 
•	Slicer Integration 
•	Insight Generation
